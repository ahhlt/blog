# XML

# 1 XML介绍

  • XML（Extensible Markup Language）可扩展标记语言。
  • 用途：
    1. 程序之间的数据传输，使用XML格式文件传送数据。现在基本改用JSON传输。
    2. 配置文件，XML格式的配置文件最普遍。
      • Spring配置文件：beans.xml；
      • MyBatis配置文件：xxMapper.xml；
      • Maven配置文件：pom.xml；
      • tomcat配置文件：server.xml、web.xml；
    3. 小型数据库（存储数据），使用xml文件直接读取数据。
  • XML没有预定义的标签，所有标签由用户自己定义。

# 3 XML语法

  • XML标签必须成对出现，一个开始标签(<标签>)和一个结束标签(</标签>)；

  • XML标签严格区分大小写，<P>与<p>是两个不同的标签；

  • XML标签必须正确嵌套，不允许交叉嵌套；

  • 注释：

    <!-- This is a comment -->
    1

  • 特殊字符：

    语法 符号 含义
    &lt; < 小于
    &gt; > 大于
    &amp; & 和号
    &apos; ' 单引号
    &quot; " 双引号

# 4 XML元素

  • XML元素：由标签和标签内包含的内容组成；
  • XML文档有且仅有一个根元素；
    • 根元素包含文档中其它所有元素；
    • 根元素的起始标记放在最前，结束标记放在最后；
  • XML标签命名规则：
    • 可以含字母、数字以及其他的字符；
    • 不能以数字或者标点符号开始；
    • 不能包含空格、冒号(:)；
    • 尽量避免**-、.以及:**。

# 5 XML属性

  • XML属性用法：<元素名 属性名 = "属性值">，属性提供元素的额外信息；

    <file type="gif">computer.gif</file>
    1

  • 标签的属性值必须加双引号("")；

  • 属性值不能包含**&**字符；

  • 属性值中包含单引号(')，就使用双引号包围(")；

    <tag name='George "Shotgun" Ziegler'></tag>
    1

  • 属性值中包含双引号(")，就使用单引号包围(")。

    <tag name="George 'Shotgun' Ziegler"></tag>
    1

# 7 XML解析技术

# XML解析原理

  • XML与HTML相同，都是标记型文档，都可以使用dom技术解析；

  • document对象表示的是整个文档，xml文档被映射为xml dom对象；

# XML解析技术

  • DOM解析和SAX解析都过时了
  • 第三方的XML解析技术
    • jdom在DOM基础上进行封装；
    • dom4j对jdom进行封装；

# DOM4J

  • DOM4J用于解析/处理XML文档；
  • DOM4J使用接口和抽象基类，是一个功能强大、性能优异的Java XML API；
  • 使用DOM4J，需要下载DOM4J相应的jar包，将jar包引入项目工程；

# 使用DOM4J解析XML文件

  • 将XML文件映射为一个DOM对象Document，通过DOM对象对XML文件内的元素/标签进行操作

# 遍历所有节点

  • 第一步：加载XML文件
  • 第二步：获取根节点
  • 第三步：获取根节点下的所有子节点（List集合）
  • 第四步：遍历List集合
/**
* 遍历XML文件中的student节点
*/
@Test
public void listXML() throws DocumentException {
    // 获取一个解析器
    SAXReader reader = new SAXReader();
    Document document = reader.read("src/students.xml");
    // 获取xml文件根节点rootElement
    Element rootElement = document.getRootElement();
    // 获取rootElement下的所有student节点
    List<Element> students = rootElement.elements("student");
    System.out.println(students.size());
    for (Element student : students) {
        // 获取student节点的name
        Element name = student.element("name");
        Element age = student.element("age");
        Element resume = student.element("resume");
        Element gender = student.element("gender");
        System.out.println(name.getText() + " " + age.getText() + " " + resume.getText() + " " + gender.getText());
    }
}

# 增加节点

/**
* 增加子节点(元素)
*/
@Test
public void addElement() throws DocumentException, IOException {
    // 读取XML文件
    SAXReader reader = new SAXReader();
    Document document = reader.read("src/students.xml");
    // 创建子节点student
    Element student = DocumentHelper.createElement("student");
    // 添加属性
    student.addAttribute("id","01");
    // 创建孙子节点name、gender、age
    Element name = DocumentHelper.createElement("name");
    name.setText("赵帅博");
    Element gender = DocumentHelper.createElement("gender");
    gender.setText("男");
    Element age = DocumentHelper.createElement("age");
    age.setText("24");
    // 将所有孙子节点添加给子节点
    student.add(name);
    student.add(gender);
    student.add(age);

    // 将子节点添加给根节点
    document.getRootElement().add(student);
    // 输出，注意编码设置为utf-8
    OutputFormat output = OutputFormat.createPrettyPrint();
    output.setEncoding("utf-8");

    // 更新XML文件
    XMLWriter writer = new XMLWriter(new FileOutputStream(new File("src/students.xml")));
    writer.write(document);
    writer.close();
}

# 删除节点

/**
* 删除节点
*/
@Test
public void delElement() throws DocumentException, IOException {
    // 读取XML文件
    SAXReader reader = new SAXReader();
    Document document = reader.read("src/students.xml");
    // 获取根节点
    Element rootElement = document.getRootElement();
    // 获取要删除的节点
    Element gender = rootElement.element("student").element("gender");
    // 通过待删除节点的父亲节点删除（必须）
    gender.getParent().remove(gender);

    // 更新XML文件
    // 输出，注意编码设置为utf-8
    OutputFormat output = OutputFormat.createPrettyPrint();
    output.setEncoding("utf-8");
    XMLWriter writer = new XMLWriter(new FileOutputStream(new File("src/students.xml")));
    writer.write(document);
    writer.close();

}

# 修改节点内容

/**
* 修改节点
*/
@Test
public void modifyElement() throws DocumentException, IOException {
    // 读取XML文件
    SAXReader reader = new SAXReader();
    Document document = reader.read("src/students.xml");
    // 获取节点
    Element name = document.getRootElement().element("student").element("name");
    // 重新设置节点的值
    name.setText("杜沛燕");
    Element age = document.getRootElement().element("student").element("age");
    age.setText("0");

    // 更新XML文件
    // 输出，注意编码设置为utf-8
    OutputFormat output = OutputFormat.createPrettyPrint();
    output.setEncoding("utf-8");
    XMLWriter writer = new XMLWriter(new FileOutputStream(new File("src/students.xml")));
    writer.write(document);
    writer.close();
}
