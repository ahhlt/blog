# CSS

# 1 CSS介绍

  • CSS (Cascading Style Sheets)：层叠样式表；

  • CSS可以同时对多个HTML元素的样式进行设置，提高了开发效率；

  • CSS样式可以直接写在HTML文件内部，也可以写在外部的.css文件中；

  • CSS样式的3种使用方式：

    1. 元素内部使用style属性定义样式；

      <!--方式1：直接在标签中添加style属性-->
<div style="color: blue; background-color: beige">
    hello,赵帅博 方式1
</div>

    <style type="text/css">
        h1{
            color: red;
            background-color: aqua;
        }
    </style>
</head>
<body>
    <h1>hello,赵帅博 方式2</h1>

# 3 CSS选择器

# 元素选择器

  • 元素选择器，又称标签选择器、类型选择器；

  • 通过指定标签来修改样式，只要是被选中的变迁都会被css样式修饰，选择所有相同标签；

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>元素选择器</title>
    <style type="text/css">
        /*通过指定标签来修改样式*/
        /*标签选择器会修改所有被选择标签的样式*/
        h1{/*只要是h1标签的都会被该css样式修饰*/
            width: 50%;
            height: 100px;
            color: red;
            border: 1px solid rebeccapurple;
            text-align: center;
            margin-right: auto;
            margin-left: auto;
        }
        p{/*只要是p标签的都会被该css样式修饰*/
            width: 50%;
            height: 100px;
            color: blue;
            border: 1px dashed aqua;
            text-align: center;
            margin-right: auto;
            margin-left: auto;
        }
    </style>
</head>
<body>
<h1>安徽理工大学</h1>
<h1>合肥智慧环境院</h1>
<p>hello,安徽理工大学</p>
<p>hello,你好</p>
</body>
</html>

# ID选择器

  • 根据id属性的值锁定标签，id是标签的唯一标识，因此ID选择器只选择一个标签；

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>id选择器</title>
    <style type="text/css">
        /*通过自己定义的id名称来指定修饰对应的标签*/
        #css1{/*css1只修饰id名称为css1的标签的样式*/
            color: aqua;
            background-color: blue;
        }
        #css2{/*css2只修饰id名称为css2的标签的样式*/
            color: blue;
            background-color: green;
        }
    </style>
</head>
<body>
<!--1.首先要给需要修饰的标签设置id-->
<div id="css1">
    hello,赵帅博
</div>
<div id="css2">
    hello,赵帅博
</div>
</body>
</html>

# 类选择器

# 选择器优先级

  • 层叠样式由三种，内联样式、内部样式表以及外部样式表
    • 内联样式（在HTML元素内部）：在标签中使用style属性指定样式；
    • 内部样式表（在<head>标签内部）：在<head>标签内部使用style标签指定样式；
    • 外部样式表：单独创建.css文件，再使用<link>标签引入到html；
  • 优先级：内联样式 > 内部样式表 > 外部样式表

# CSS文本

  • 文本居中与div块居中

    • text-align：设置文本排列方式，center文本居中，left左对齐，right右对齐；
    • margin-right和margin-left：同时为auto时实现div居中；
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>div居中与文本居中</title>
    <style type="text/css">
        div{
            width: 50%;
            height: 100px;
            background-color: beige;/*背景颜色*/
            color: blue;
            border: 1px dashed red;
            font-size: 20px;/*字体大小*/
            font-weight: bold;/*是否加粗*/
            font-family: 微软雅黑;/*字体类型，要求电脑内安装有对应的字体库*/
            /*margin-right和margin-left都设置为auto就是div左右居中*/
            margin-left: auto;
            margin-right: auto;/*div居中*/
            text-align: center;/*center 文本居中，left左对齐，right右对齐*/
        }
    </style>
</head>
<body>
<div>安徽理工大学 赵帅博</div>
</body>
</html>

# CSS字体

CSS 字体属性定义文本的字体类型、大小、加粗、风格（如斜体）和变形（如小型大写字母）。

  • font-family属性：字体类型

  • font-style属性：字体风格，如斜体、文本倾斜

  • font-weight属性：文本粗细，bold加粗

  • font-size属性：指定字体大小

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>字体样式</title>
    <style type="text/css">
        div{
            width: 50%;
            height: 100px;

            border: 1px dashed red;

            font-family: 微软雅黑;/*字体类型，要求电脑内安装有对应的字体库*/
        }
    </style>
</head>
<body>
<div>安徽理工大学 赵帅博</div>

# CSS链接

  • 链接的四种状态：

    • link：普通的、未被访问的链接

    • visited：已访问的链接

    • hover：鼠标指针位于链接的上方

    • active：正在被点击的链接

      a:link {color:#FF0000;}		/* 未被访问的链接 */
a:visited {color:#00FF00;}	/* 已被访问的链接 */
a:hover {color:#FF00FF;}	/* 鼠标指针移动到链接上 */
a:active {color:#0000FF;}	/* 正在被点击的链接 */
