# JavaScript

# 1 介绍

  • JavaScript是一种轻量级的解释型脚本语言；

  • 弱类型机制，变量的数据类型可以变化；

  • JavaScript 对大小写敏感；

  • 脚本语言不用编译，浏览器会在读取代码时，逐行地执行脚本代码；

  • JavaScript的作用：

    • 读写HTML文件
    • 响应事件
    • 改变元素内容、图像、样式
    • 验证用户输入

  • 使用JavaScript的方式：HTML中的脚本必须位于<script>与</script>标签之间

    1. 方式1：在<head>或<body>标签中嵌入JS代码

      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>使用方式1-写在script标签中</title>
	<!-- JS代码一般写在head中，也可以写在body中 -->
    <script type="text/javascript">
        // 1.在head中使用script写js代码
        console.log("写在head中")
    </script>
</head>
<body>
<script type="text/javascript">
    // 2.在body中使用script写js代码
    console.log("写在body中")
</script>
</body>
</html>

# 2 变量与数据类型

  • 变量是存储数据的容器；

  • var关键词：声明变量；

  • 变量的数据类型，取决于所赋值的数据类型，且随着赋值类型的变化而变化；

  • 5种数据类型：

    1. number：数字类型
    2. string：字符串类型
    3. object：对象类型
    4. boolean：布尔类型
    5. function：函数类型

    3. NaN：非数值类型，如number类型与string类型相乘

# 定义数组

  2. 方式2：先声明，再通过数组索引赋值；
  3. 方式3：var 数组名 = new Array("元素1", "元素2", ..., "元素n");
  4. 方式4：先创建数组对象，再指定参数

# 细节说明

  1. JS中函数的重载会覆盖前一次的定义；
  2. 函数的arguments隐形参数，作用域在function函数内；
    • 隐型函数：function函数形参列表中不需要定义参数，直接获取传入实参的值；
    • 类似于Java中的可变参数，传入几个实参，arguments的长度就为多长；
    • arguments隐形参是一个数组，可以按数组进行操作
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>函数使用细节</title>
    <script type="text/javascript">
        // 1.JS中函数的重载会覆盖掉上一次的定义
        function f1() {
            alert("f1(),zhaoshuaibo");
        }
        // 重载直接覆盖
        function f1(name) {
            alert("f2()," + name);
        }
        // 调用
        f1();

        // 2.函数的arguments隐形函数（作用域在function函数内）
        // (1)隐形参数：在function函数中不需要定义形参，也可以直接传入实参
        // (2)隐形参数底层机制：function(arguments)，arguments是数组可以遍历，
        // (3)function(arguments)类似于java中的可变参数fun(int ...args)
        function f2() {
            // 输出arguments数组
            console.log("arguments= ", arguments);
            // arguments是数组，可以按数组操作
            console.log("arguments长度= " + arguments.length);
            alert("f2()...");
        }
        f2(10, 20, 30);

        // 3.如果函数定义了形参，但
        // (1)传入的实参个数比形参多：先与定义的形参匹配，再将所有的实参赋给argument
        // (2)传入的实参个数比形参少：未匹配到实参的形参的值为undefined
        function f3(n) {
            console.log("n= " + n);
            console.log("arguments= ", arguments);
        }
        f3(100, 200, 300);
        
    </script>
</head>
<body>

</body>
</html>

# 7 事件

# 事件介绍

  • 事件注册（绑定）：将事件与JavaScript代码绑定，当事件响应/触发后，浏览器就执行对应的JavaScript代码。
  • 静态注册：通过HTML标签的事件属性注册，将事件属性的值赋为事件触发后的代码
  • 动态注册：
    1. 先获取标签的DOM对象；
    2. 再通过，**标签DOM对象.事件名 = function(){}**来绑定代码。

# onload页面已加载

  • onload事件会在用户进入页面时被触发

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>事件onload加载完成事件</title>
    <script type="text/javascript">

      // 静态绑定：将事件写在标签内
      // 定义一个函数，当触发事件时执行该函数
      function sayOK(){
        alert("静态绑定 onload事件sayOK");
      }

      // 动态注册：使用DOM对象绑定对应事件
      // 1.window现有的dom对象，包含很多属性和函数，
      // 2.window.onload表示页面被 加载完毕
      // 3.后面的function(){}指加载完毕后要执行的代码
      window.onload = function (){
        alert("动态绑定 onload事件")
      }
    </script>
</head>
<!--静态注册，当body加载完成后触发onload事件，执行sayOK()方法-->

# onclick点击

  • onclick事件当鼠标点击指定对象时被触发

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>事件onclick鼠标点击</title>
    <script type="text/javascript">
      // 静态绑定
      var f1 = function (){
        alert("这是静态绑定...");
      }
      // 动态绑定
      // 当页面加载完毕以后再动态绑定事件
      window.onclick = function (){
        // 1.先拿到id=btn2的button对应的DOM对象
        var btn2 = document.getElementById("btn2");
        // 2.通过DOM对象绑定onclick事件
        btn2.onclick = function (){
          alert("这是动态绑定...")
        }
      }

    </script>
</head>
<body>
<button onclick="f1()">静态绑定事件</button>
<button id="btn2">动态绑定事件</button>
</body>

# 案例2-getElementByName()

  • 复选框实现全选、全不选、反选功能

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>DOM对象getElementByName方法</title>
    <script type="text/javascript">
        // 静态绑定事件
        function selectAll() {
            //先获取标签对应的DOM对象，通过getElementsByName()方法
            //返回一个NodeList集合，包含所有name="sport"的标签
            var sports = document.getElementsByName("sport");
            // 遍历sports集合，修改每个标签的值
            for (var i = 0; i < sports.length; i++) {
                sports[i].checked = true;
            }
            sports.checked.value = "checked"
        }

        function selectNone() {
            var sports = document.getElementsByName("sport");
            // 遍历sports集合，修改每个标签的值
            for (var i = 0; i < sports.length; i++) {
                sports[i].checked = false;
            }
            sports.checked.value = "checked"
        }

        function selectReverse() {
            var sports = document.getElementsByName("sport");
            // 遍历sports集合，修改每个标签的值
            for (var i = 0; i < sports.length; i++) {
                //反选
                if (sports[i].checked == true){
                    sports[i].checked = false;
                }
                if (sports[i].checked == false){
                    sports[i].checked = true;
                }
                // 一条语句解决
                //sports[i].checked = !sports[i].checked;
            }
            sports.checked.value = "checked"
        }
    </script>
</head>
<body>
<input type="checkbox" name="sport" value="zq" checked="checked">足球
<input type="checkbox" name="sport" value="tq">台球
<input type="checkbox" name="sport" value="ppq">乒乓球
<button onclick="selectAll()">全选</button>
<button onclick="selectNone()">全不选</button>
<button onclick="selectReverse()">反选</button>
</body>
</html>
