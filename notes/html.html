# HTML

该文档是笔者（爱喝胡辣汤）学习HTML时积累的学习笔记，目的是方便复习回顾。由于笔者重心在学习后端开发，因此只涉及到HTML的重点、常用内容。

# 1 HTML简介

HTML是用来描述网页的一种语言。

  • HTML(**H**yper **T**ext **M**arkup **L**anguage)：超文本标记语言，使用一套标记标签来描述网页，标签又称为网页的元素。
  • HTML文本：由HTML标签组成，可以包括文字、图像、链接、表格等。
  • HTML标签：通常由尖括号**<标签>包围且成对出现，如<html></html>、<p></p>**都是标签对。
    • 标签对：由开始和结束标签组成，第一个为开始标签，后面是结束标签；
    • 单标签：单标签只有开始标签，如**<br>、<hr>**。
  • HTML结构：HTML由head和body两部分组成，分别由标签**<head></head>和<body></body>**控制。
    • <head></head>：HTML头信息，可以写入CSS文件的引入，JavaScript代码；
    • <body></body>：网页的主体，存放页面内容。

第一个HTML文件：

<!DOCTYPE html> <!-- 文件类型说明 -->
<html lang="en"> <!-- 整个HTML文件的开始 -->
<head> <!-- HTML头信息 -->
    <meta charset="UTF-8"> <!-- 指定文件的字符集 -->
    <title>第一个HTML文件</title> <!-- 网页的标题 -->
</head>
<body> <!-- 网页的主体部分，存放网页的内容，如文本、图片、视频等 -->
<!-- HTML的注释 -->
</body>
</html> <!--HTML文件的结束-->

网页的组成：

  • 结构(HTML)：HTML是网页内容的载体，搭建整个网页的框架结构，内容可以是文字图片视频等；
  • 表现(CSS)：CSS样式是表现，是网页的外衣，控制着字体颜色、背景、边框等；
  • 行为(JavaScript)：JavaScript用于实现网页的特效，如图片轮换，鼠标点击弹窗等。

# 图像标签

  • HTML 图像通过**<img>**标签进行定义；

  • 图像标签可以将图片显示在页面上；

  • 基本用法：<img src="图像路径" width="宽度px" height="高度px" />

    • **src**属性：指定图片的路径；
      • 相对路径： **.表示当前文件所在的目录，..**表示当前文件所在的上一级目录；
      • 绝对路径：http://IP地址:port/工程名/资源路径；
    • **width属性：指定图片宽度，height**属性：指定图片高度，只指定一个属性时另一个属性会跟随变化；
    • **border**属性：设置图片边框的粗细；
    • **alt**属性：指定当图片路径找不到图片时，用来代替显示的文本内容。
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>演示图像标签</title>
</head>
<body>
<h1>图片标签的演示</h1>
<p>
    这是一个段落标签<br>
    <img src="./img/img1.jpg" width="200">
    <img src="./img/img1.jpg" width="200" height="350">
    <img src="./img/img1.jpg" width="200" border="10px">
    <img src="./img/img2.jpg" alt="图片未找到">
</p>
</body>
</html>

# 有序列表

  • **<ol>**标签：ordered list，定义有序列表；
  • **<li>**标签：list，列表项；
  • **type**属性：指定列表项前面的符号形状；
    • 1：阿拉伯数字1，2，3，...；

# 表格标签

  • 表格由**<table>**标签来定义；

# 3 HTML元素

  • HTML元素：由开始标签、结束标签以及标签之间的内容组成；
    • 如**<p> This is a paragraph </p>**；
  • HTML元素以开始标签起始，结束标签终止；
  • 元素的内容是开始标签与结束标签之间的内容；
  • 空元素（无内容）直接在开始标签中关闭，如<br />；
  • HTML标签对大小写不敏感，但建议使用小写。

实例：

  1. **<html>**元素定义了整个HTML文档；

# 4 HTML属性

  • HTML属性定义在开始标签中；
  • 属性和属性值对大小写不敏感，建议使用小写；

# 5 HTML表单

  • 表单由**<form>**标签来定义；
  • 表单元素指的是不同类型的input元素、复选框、单选按钮、提交按钮等等；

  • 单选、复选都需要添加value属性，以便将选择结果发给服务器；

# GET请求与POST请求

GET请求：请求数据显示在浏览器地址栏，不安全，且数据长度有限；

POST请求：请求数据保存在http请求体中，相对安全，数据长度理论上不限，但也不能太大；

# div标签

  • **<div>**标签把HTML文档分为独立的、不同的部分；
  • <div>标签是一个块级元素，不同的块之间以换行分隔；
<!DOCTYPE html>
<html lang="en">

</head>
<body>

hello,world
<div>
    <h3 style="color: red">This is a H3</h3>
    <a href="http://www.baidu.com" target="_blank">goto百度</a>
</div>
</body>
</html>
